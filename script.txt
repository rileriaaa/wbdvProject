[PROJECT OVERVIEW]
"Scholar's Den is a student-focused online marketplace designed
for buying and selling textbooks, school supplies, and educational
materials. The platform directly addresses the challenge students
face in finding affordable and quality academic resources.

Instead of going to multiple stores or social media groups,
Scholar's Den gives students one centralized platform where they
can browse listings, purchase items, and even sell their own
materials — all within a trusted student community."

[DESIGN & FEATURES]
"The platform is built as a fully frontend-only application using
Next.js, TypeScript, and Tailwind CSS. Since there is no backend
or database, all dynamic functionality — including user
authentication, cart management, and order tracking — is
simulated through React state, static JSON data, and localStorage.

Key features of Scholar's Den include:

  • A landing page with featured listings, category browsing,
    and a search bar accessible from every page

  • A Browse and Search page where users can filter listings
    by category, price range, and item condition

  • An Item Detail page showing photos, ratings, seller
    information, and related items

  • A full Cart and Checkout flow with shipping options,
    payment methods, and order confirmation

  • A Messaging system so buyers can talk to sellers directly

  • A Profile section covering purchase history, wishlist,
    account settings, and a full Seller Dashboard

  • Protected routes — pages like Cart, Profile, and Messages
    are only accessible to logged-in users

  • Dark mode support across all pages"

[WHAT WE ACCOMPLISHED TODAY]
"For today's progress, our team has completed the following:

  • All 27 pages of the website have been designed and built
  • The sitemap and wireframes have been finalized
  • The project proposal document has been completed and submitted

[HOW TO NAVIGATE]
"Let me walk you through how to navigate the site.

  [Show on screen while speaking]

  • From the Landing Page, you can browse featured listings,
    use the search bar, or filter by category using the pills
    below the navbar

  • Clicking any product takes you to the Item Detail Page,
    where you can add it to your cart or wishlist — though
    these actions require you to be logged in

  • The Sign Up page lets you register as a buyer, seller,
    or both. The Log In page accepts any credentials for
    demo purposes and redirects you back to where you were

  • Once logged in, the navbar shows your cart, notifications,
    and a user dropdown with links to your profile and
    seller dashboard

  • On mobile, the hamburger icon opens a slide-in drawer
    from the right side with all navigation links

  • The Browse page has a filter sidebar for category, price
    range, and condition, with active filter chips shown above
    the product grid"

[DESIGN PROCESS]
"We started working on the visual design after the wireframes
were approved. Our design direction for Scholar's Den is a
clean, modern editorial style — white backgrounds, sharp
typography, and a strong accent color.

For the design application, we used Figma to plan the visual
system before implementation. The actual UI was then built
directly in code using Tailwind CSS v4.

Our color system uses CSS variables for both light and dark
mode. The primary accent color is a warm orange-red — hex
#d4522a — which gives the platform a distinct identity that
stands out from typical blue-heavy e-commerce sites.

For typography, we used DM Sans as the primary font for its
clean, modern feel, and Instrument Serif as a display font
for headings, giving the site a refined editorial quality.
Both fonts are loaded from Google Fonts."

[MOCKUPS]
"  [Show the actual website or the PDF wireframes]

For the mockups, all 27 pages were designed with full
responsiveness in mind. Every page works across desktop,
tablet, and mobile breakpoints.

Key design decisions include:

  • The dark hero sections on the Landing Page, About Page,
    Login and Signup use a fixed dark surface color that
    stays dark in both light and dark mode, while the rest
    of the UI adapts to the user's theme preference

  • Product cards use a hover elevation effect with a subtle
    shadow and upward translate to give depth and interactivity

  • The mobile navigation uses a slide-in drawer from the
    right side with a blurred backdrop, instead of pushing
    the page content down

  • Form pages like Login, Signup, and Checkout use a
    two-column split layout on desktop — a dark branded
    panel on the left and the form on the right

All icons used in the interface are emoji-based, requiring
no external icon library and keeping the bundle size small."

[TECH STACK & IMPLEMENTATION]
"Scholar's Den was built using Next.js with the App Router,
TypeScript for type safety, and Tailwind CSS v4 for styling.

Since this is a frontend-only application, all data is managed
through static JSON files and React state. Key technical
implementations include:

  • Next.js App Router with file-based routing — pages map
    directly to their URL path, and dynamic routes like
    /item/[id] and /seller/[id] handle individual listings
    and seller profiles

  • Middleware-based route protection — unauthenticated users
    are automatically redirected to the login page when
    accessing protected routes like Cart, Messages, and Profile

  • localStorage is used to persist simulated session data
    such as the logged-in user, cart contents, and wishlist
    items across page refreshes

  • Dark mode is implemented using Tailwind's dark variant
    combined with a class toggle on the root element,
    with a fixed dark surface for hero sections regardless
    of theme

  • The mobile navigation drawer is implemented as an
    absolutely-positioned overlay — it slides in from the
    right without affecting the document layout or pushing
    page content"
